Register a Node:
curl -X POST -H "Content-Type: application/json" -d '{"node_id": "your_node_id"}' http://localhost:5000/nodes/register

Mine a Block for a User:
curl http://localhost:5000/mine_block/efe123

Display Blockchain for a User:
curl http://localhost:5000/get_chain/efe123

Check Blockchain Validity for a User:
curl http://localhost:5000/valid/efe123

Add a Transaction:
curl -X POST -H "Content-Type: application/json" -d '{"sender": "sender_id", "recipient": "recipient_id", "amount": 5}' http://localhost:5000/add_transaction/efe123

Resolve Conflicts:
curl http://localhost:5000/nodes/resolve
